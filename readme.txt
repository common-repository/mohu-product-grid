== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Moo Woocommerce Product Grid</strong>" activate it.<br />

After activate plugin you will see "Moo Grids" menu at left side on WordPress dashboard click "Moo Grids" and use the options field "Moo Grids"<br />

<br />
<strong>How to use on page or post</strong><br />
When use the shortcode on your page sections options setup done please publish Post Grid as like post or page<br />

then paste this shortcode anywhere in your page to display grid<br />
